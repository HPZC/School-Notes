# 第十章 物质的运输和排泄

## 10.1 血液和血型

### 10.1.1 血液的组成和功能

- 血液：红色粘稠液体，占成人体重的7-8%
- 若将人类的新鲜血液放入盛有少量抗凝剂[^抗凝剂]的试管内静置一段时间，试管中会分层
  - 上层：**血浆** - **透明、淡黄色**的液体
    - 占血液总量的55%，其90%以上是水，其余是血浆蛋白、蛋白质、葡萄糖、无机盐、废料[^废料]等
    - 血浆的主要功能是**运载血细胞、运输营养物质和二氧化碳等代谢产物**
    - 血清：血浆 - 蛋白质
    
  - 下层：**血细胞** - 呈暗红色
  
- **血液**就是由**血浆和各种血细胞组成**

#### 血细胞

- ##### 红细胞

  - 血细胞中**数量最多**
  - 成熟的红细胞**没有细胞核**、呈**两面凹的圆盘形**
  - 有**红色含铁的蛋白质**——**血红蛋白**，使**血液呈红色**
    - 血红蛋白在氧浓度高的地方容易跟氧结合，在氧浓度低的地方容易跟氧分离
    - 血红蛋白帮助人体**运输氧气**和部分二氧化碳
  - 贫血：血液里**红细胞的数量过少**，或红细胞中**血红蛋白过少**

- ##### 白细胞

  - 比红细胞大，**有细胞核**，**数量最少**
  - 白细胞可以**吞噬病菌**，人体中含有多种白细胞以**抵抗病菌的侵入**

- ##### 血小板

  - 血细胞中**体积最小**，没有细胞核，形状不规则
  - 人体受伤流血时，血小板会在伤口聚集成团，促进**止血和加速血液凝固**
  - 过少：影响止血功能、危及生命；过多：易造成血栓


- 人体内的血细胞在不断衰老、死亡和再生

  - 红细胞大约六十天会更新一半

- 显微镜下的视野中最容易观察到的细胞：红细胞

- 骨内的**红脊髓**具有造血功能，能**产生新的血细胞**

- 血细胞含量受影响的一些因素

  - | 疾病 |         血细胞含量的改变         |
    | :--: | :------------------------------: |
    | 炎症 |       白细胞数值超过正常值       |
    | 贫血 | 红细胞计数及血红蛋白数值都会下降 |
    | 血栓 |            血小板过多            |

- 血液的功能

  - **输送氧、各种营养物质及二氧化碳等代谢产物**
  - **起防御保护作用**
  - **调节体温**

### 10.1.2 血型与输血

- 最常用的血液分类法是**ABO血型系统**

  - 将人类血型分为A型、B型、AB型、O型四种

- 血型是由**红细胞膜上抗原的类型**决定的

- **血浆**中会有**与抗原相对应的抗体**

- | 血型  | 红细胞上的抗原  |  血浆中的抗体   |
  | :---: | :-------------: | :-------------: |
  | A 型  |     A 抗原      |     B 抗体      |
  | B 型  |     B 抗原      |     A 抗体      |
  | AB 型 | A 抗原 & B 抗原 |       无        |
  | O 型  |       无        | A 抗体 & B 抗体 |

- 某种抗原的红细胞和抗该抗原的抗体相遇，就会发生凝集[^凝集]反应，引起红细胞大量凝集

- 人的血量需保持相对稳定以保证生理活动的正常进行

  - 失血量不超过血液总量的的10%(400ml)：血浆和血细胞可以在短期内恢复正常，对身体健康没有太大的影响
  - 失血量超过血液总量的30%(1200ml)：严重影响人体生命活动，甚至危及生命，须通过输血以补充血量

- 输血

  - 输血时须先进行**交叉配血实验**
    - 交叉配血实验考虑供血者的红细胞上抗原是否会和受血者血浆中的抗体发生凝集作用
    - 主侧：受血者血清加供血者红细胞悬液
    - 次侧：供血者血清加受血者红细胞悬液
    - 交叉配血试验的过程中，若主侧的结果是没有凝集作用需继续进行次侧
    - **O型血**的**红血球因不含抗原**而成为**全适的输血者**
    
  - **输血时应该以输同型血为原则**


## 10.2 血管和心脏

- **心脏和血管**构成了**人体血液循环的管道**
- 血液能不停按照一定方向流动

### 10.2.1 血液流动的管道——血管

- ![三种血管的关系](https://i.ibb.co/FwLrkrr/image.png)

- 血管分为**动脉、静脉和毛细血管**三种

- |      |                   动脉                   |                             静脉                             |                           毛细血管                           |
  | ---- | :--------------------------------------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
  | 功能 |        把血液送到全身各器官的血管        |               把血液从全身各器官送回心脏的血管               |              血液和组织细胞间进行物质交换的场所              |
  | 特征 | 管壁厚、弹性大、管内血流速度快、血压最大 | 管壁薄、弹性小、管腔大、管内血流速度慢、血压最小。管腔内有半月形的静脉瓣，防止血液倒流 | 管壁极薄，仅有一层上皮细胞构成，管径极细，一般只允许红细胞排成单行通过 |
  |      |     颈部：颈动脉；腕部：桡 (ráo)动脉     | 皮肤下的“青筋”就是静脉。抽血、注射疫苗都在静脉，静脉靠近皮肤，血流速度较慢，安全性较高 | 连接在细小的动脉和静脉之间的血管、毛细血管血压 > 静脉血压、毛细血管流速 < 静脉流速 |
  

### 10.2.2 血液运输的动力器官——心脏

- **血液流动的动力**来自**心脏**
- 通过心肌**不断地收缩和舒张**，推动**血液**在**心脏和血液**中**沿固定的方向不间断地流动**
- 心脏位于**胸部的中部偏左下方**，大小和**拳头**差不多
- ![心脏结构](https://i.ibb.co/3MDYTQB/image.png)
- 心脏内部的四个腔：**左心房、左心室**、**右心房、右心室**
- **同侧的心房和心室相通**，左右心房和左右心室互相隔开
- 主动脉管壁比肺动脉管壁厚
- **心室与动脉**相连，**心房与静脉**相连
  - **左心室连主动脉**，**左心房连肺静脉**
  - **右心室连肺动脉**，**右心房连上下腔静脉**
- **心房和心室**之间及**心室和动脉**之间有**瓣膜**以**控制血液向一个方向流动**

- ![心脏内部血流方向](https://i.ibb.co/7C3bFtq/image.png)
- 当左右**心房收缩**，左右**心室舒张**时
  - **心房的血液进入心室**
  - 心房和心室之间的**房室瓣**会**防止血液倒流回心房**
- 当左右**心室收缩**，左右**心房舒张**时
  - **左心室**会把**血液压入主动脉**输送到**全身**
    - 被毒蛇咬伤时应该包扎在近心端，减少或者避免毒素进入血液循环的可能
  - **右心室**会把**血液压入肺动脉**送往**肺部**
- **心脏的搏动**推动着**血液流动**，是**血液运输的动力器官**
  - 颈动脉和桡动脉的搏动与心脏的跳动有关，**心脏每跳动一次，颈动脉和桡动脉就相应的搏动一次**

### 10.2.3 心脏和血管的保护

- **心血管疾病**是一系列合计循环系统的疾病，如：动脉硬化、冠心病、脑血栓、脑溢血
- 原因：日常生活、饮食习惯、精神状态
  - 长期精神紧张
  - 食物中脂肪过多
  - 嗜烟
  - 嗜酒
  - 缺乏体育运动

#### 动脉硬化

- 血液中**脂肪类物质在动脉壁上沉积**引起的
- 使**动脉壁增厚，弹性降低，管腔狭窄**
- 血液中低密度脂蛋白[^低密度脂蛋白]和总胆固醇水平高的人容易发生动脉硬化

#### 冠心病

- **心肌运动**需要**能量**，**冠状动脉供应**心脏本身**营养**需要的动脉
- **冠心病**：**冠状动脉发生病变**，如动脉硬化、管腔变窄、导致心肌缺血
- **冠心病严重**时会引起**心绞痛**甚至发生**心肌梗死**
- 若**冠状动脉被阻断**，**心跳**很快就会**停止**

#### 中风

- 中风：**脑动脉硬化**导致动脉栓塞或血管破裂引发的**脑溢血**
- 中风症状
  - 猝然昏倒、不省人事
  - 口角歪斜、语言不利
  - 半生不遂
- **中风**同**冠心病、癌症**并列为**威胁人类健康的三大疾病**
  - 中风发病率高、死亡率高、致残率高、复发率高、并发症多

### 生活习惯对心血管健康的影响

- 经常参加体育锻炼的人
  - 心肌发达，心脏搏动有力，动脉口径会增大，管壁的弹性得到改善
  - 每搏输出量增大，只要较少的心脏搏动次数就能达到一般人的每分钟输出量，心跳速率也就比一般人慢
- 心脏较弱的人
  - 需加快心跳速率以提高心脏的每分钟输出量
  - 心肌无法得到充分休息，容易疲劳，因此不能坚持长时间的剧烈运动

## 10.3 血液循环

- **人的循环系统**分为**血液循环系统**和**淋巴循环系统**
- **血液循环系统**：**血液**、**血管**和**心脏**组成
- 血液循环：心脏跳动促使血液在心脏和全部血管所组成的管道中沿固定方向循环流动

#### 10.3.1 体循环和肺循环

- 人体血液循环：可分为**体循环**和**肺循环**
- 体循环
  - 左心室 ——> 主动脉 ——> 各级动脉 ——> 毛细血管 ——> 各级静脉 ——> 上下腔静脉 ——> 右心房
  - 血液通过毛细血管网和身体的各部分的组织细胞进行物质交换
    - 把运来的氧气和养料供给细胞利用，同时把细胞产生的二氧化碳等废物运走
  - 从左心室射出的鲜红色充氧血，经过体循环变成暗红色的缺氧血
    - 充氧血：含氧量较多的血液，呈鲜红色
    - 缺氧血：含氧量较少的血液，呈暗红色
- 肺循环
  - 右心室 ——> 肺动脉 ——> 肺毛细血管 ——> 肺静脉 ——> 左心房
  - 血液在肺部毛细血管网进行气体交换
    - 血液排出二氧化碳获得氧气
  - 从右心室射出的暗红色缺氧血，经过肺循环变成鲜红色的充氧血
  - 由于血液经过肺部毛细血管时压力已经很低，假如不回流心脏加压，直接流向身体其他部分是很难的，这会造成组织细胞得不到氧气和营养物质的补充
- 血液循环的功能
  - 不断地将氧气、营养物质和激素等运送到全身各个组织器官
  - 将组织器官产生的二氧化碳和其他代谢废物运送到肾脏等排泄器官以排出体外
  - 保证机体各项生理活动的正常进行

### 10.3.2 心率、脉搏和血压

#### **心率**

- **心脏每分钟跳动的次数**
- 健康成年人安静时心率约为**每分钟75次**

#### **脉搏**

- 体表可触摸到的动脉搏动
- **心脏**每次**收缩**都会**产生很大的压力**，这个**压力会沿着动脉向前推进血液**，使**所有动脉都受到压力**
- **脉搏和心率是相同的**
- **中医**在诊断时会根据**腕部桡动脉**的脉相来诊断疾病
  - **脉相**：脉搏的快慢、强弱、深浅的情况

#### **血压**

- 血液在血管内向前流动时**对血管壁产生的压强**
- 血压是不断变化的
- 收缩压：心脏收缩时动脉血液达到的最高值
- 舒张压：心脏舒张时动脉血液达到的最低值
- 血压一般以分式表示
  - 120/80 mmHg 
    - 收缩压：120 mmHg
    - 舒张压：80 mmHg

## 10.4 淋巴循环

- 淋巴循环是血液循环的辅助部分
- 淋巴循环的动力来源：心脏、血压

### 10.4.1 淋巴系统的组成

- **淋巴系统**包括**淋巴、淋巴管和淋巴器官**
- **淋巴**：**无色透明的液体**，内含**淋巴细胞**
- **淋巴管**：**淋巴运行的管道**
  - **毛细淋巴管**：淋巴管道的**起始段**
    - **小肠绒毛**内有许多毛细淋巴管
  - **毛细淋巴管 ——> 各级淋巴管 ——> 淋巴总管 ——> 静脉**

#### 淋巴器官

##### 脾

- 位于腹腔左上方、呈紫红色，扁圆形
- 体内**最大的淋巴器官**
- 有**破坏衰老血细胞的作用**

##### 胸腺

- 位于胸骨后面
- 能分泌**胸腺激素**，促使**淋巴细胞发育成熟**

##### 扁桃体

- 位于**消化道和呼吸道的交汇处**
- 具有**淋巴结**
- 具有抗细菌、抗病毒的**防御功能**

##### 淋巴结

- 扁圆形或椭圆形的小体
- 沿着淋巴管分布，淋巴管上的膨大的部分
- 在人的颈部、腹股沟和腋窝密集
- 内有淋巴球，会过滤淋巴
- 细菌侵入人体发生感染时，淋巴结会**肿大疼痛**
  - 如喉咙发炎时，下巴颏下会有两个肿块，就是淋巴结
  - 炎症消失后是，淋巴结肿快会自然缩小

### 10.4.2 淋巴系统的功能

#### 淋巴循环

- 淋巴经各级淋巴管和淋巴结后，汇入***锁骨下静脉***（上腔静脉），**进入血液循环**
- 淋巴循环时**单向流动**而不形成真正的循环
- 淋巴循环中，**淋巴结**是**淋巴回流的重要滤器**
  - 淋巴结内的**淋巴细胞**（淋巴球）能**吞噬细菌及其他引起疾病的微生物**
  - 淋巴系统通过淋巴循环协助体液回流
  - 对机体起着防御和保护作用

## 10.5 人体的排泄

- 排泄：将**体内物质的代谢废物**，**经血液循环**由一定的途径排出体外的过程
- 排泄通过**泌尿系统、肺和皮肤**进行

### 10.5.1 泌尿系统的组成

- 主要功能：**形成尿液**和**排出尿液**
- ![泌尿系统](https://i.ibb.co/3WbspmC/image.png)
- **肾脏：*形成尿液*的器官**、过滤血液
  - ![肾的纵切面图](https://i.ibb.co/1R8cySX/image.png)
  - 位于人**腰后部脊柱两侧，左右各一个**
  - 形似**蚕豆**，呈**红褐色**
  - 肾皮质位于肾的浅层，呈红褐色
  - 肾髓质位于肾皮质的深部，色较淡
  - **肾盂**（yú）位于肾的内侧凹陷处，**下端连着输尿管**
  - **血液**流经肾脏时会**被过滤**
    - **吸收**葡萄糖、氨基酸、水分等**有用的物质**
    - **废物**如*尿素、尿酸*和*多余的水分及矿物盐*则**形成尿液**
      - **尿素**是**肝脏**形成的
    
  - 能**自动调节尿液的成分**
    - 大量饮水后尿液中水含量较高
    - 每天需喝足多的水，让体內产生的废物如尿素能及时排出体外
  
- **输尿管**：**输送尿液**的器官，上接肾盂，下连膀胱
- **膀胱**：**贮存尿液**，平滑肌构成的囊状器官
- **尿道**：**尿液从膀胱排出体外的通道**
- **排尿**
  - **排出废物**
  - **调节体内的水和无机盐的平衡**
  - **维持组织细胞的正常生理功能**

- 当**人体肾功能发生某些障碍**，**不能将血液中的尿素等代谢废物排出**时，人体就会**中毒**
  - 对于严重肾功能障碍者：**人工肾、肾移植**
    - **洗肾**（人工肾）：肾元失去功能，病患需要**血液解析**以**排出血液中的废物和杂质**

### 10.5.2 皮肤与汗液分泌

- **皮肤**通过汗液排出代谢废物
- 汗液的主要成分为水，其次为无机盐和尿素等
- **肺**也具有排泄作用

## 10.6 植物的排泄作用

### 10.6.1 二氧化碳和氧气的释放

- 植物**没有排泄系统**，但**有排泄作用**
- 植物体时刻都在进行**呼吸作用**，在光照条件下进行**光合作用**，并将呼吸作用产生的二氧化碳和光合作用制造的氧气释放到空气中
- **植物体的排泄作用**：**排出二氧化碳**和**释放多余的氧气**
  - 植物体还会通过**蒸腾作用**排出水分【植物从根部吸收的99%的水分都将排出体外】
    - 促进植物体从土壤中吸收水分
    - 促进水分和无机盐的运输
    - 使植物体有效的散热

### 10.6.2 植物的分泌物

- 某些**植物细胞在代谢过程**会合成**分泌物**
- 如糖类、挥发油、有机酸、树脂、油类、蛋白质、酶等
- 许多植物的分泌物都具有重要**经济价值**

[^抗凝剂]: 如：柠檬酸钠，防止血液凝固
[^废料]: 如：尿素、尿酸等等
[^凝集]: 在正常情况下红细胞是均匀分布在血液中的。当加入同种其他个体的血清时，有时可使均匀悬浮在血液中的红细胞聚集成团，这便是红细胞的凝集。
[^低密度脂蛋白]: 负责在血液内运载脂肪酸分子至全身供细胞使用
